Data Compliance

This dataset is reconciled monthly using the TikTok Research API Batch Compliance endpoint. Videos that have been deleted, made private, or removed by their authors are cleaned from the dataset to maintain integrity.

Dataset Fields

  • hashed_videoId — SHA-256 hashed video identifier
  • hashed_authorId — SHA-256 hashed author identifier
  • dateTime — video publication date
  • desc — video description text
  • hashtagNames — comma-separated hashtag list
  • stickersText — extracted sticker/overlay text
